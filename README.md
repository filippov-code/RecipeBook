# RecipeBook - книга рецептов
Приложение для создания, редактирования и хранения рецептов блюд на телефоне  
**Рецепт состоит из:**
+ Название
+ Описание
+ Список ингридиентов
+ Шаги, пройдя которые, рецепт будет готов

**Стек:**
+ С#
+ Xamarin
+ SQLite
